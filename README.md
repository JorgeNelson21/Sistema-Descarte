Este projeto consiste em uma aplicação web simples desenvolvida para auxiliar os usuários a encontrar pontos de descarte de materiais, registrar denúncias de descarte irregular e solicitar coleta de resíduos.

## Tecnologias Utilizadas

* **HTML** – estrutura das páginas
* **CSS** – estilização da interface
* **JavaScript** – lógica do sistema e manipulação da interface

## Fluxo do Sistema

O funcionamento do sistema segue um fluxo simples de interação com o usuário:

1. **Login ou Cadastro**
   Ao acessar o sistema, o usuário pode realizar o login informando usuário e senha ou criar uma nova conta através da tela de cadastro.

2. **Acesso ao Menu Principal**
   Após o login, o usuário é direcionado ao menu principal, onde pode escolher entre as funcionalidades disponíveis na plataforma.

3. **Escolha da Funcionalidade**
   No menu, o usuário pode:

   * Consultar pontos de descarte informando a cidade
   * Registrar uma denúncia de descarte irregular
   * Visualizar denúncias ativas registradas no sistema
   * Solicitar a coleta de materiais

4. **Preenchimento das Informações**
   Dependendo da opção escolhida, o usuário preenche os campos necessários. O sistema realiza validações para garantir que todos os campos obrigatórios estejam preenchidos.

5. **Confirmação da Ação**
   Após o envio das informações, o sistema exibe mensagens de confirmação indicando que a denúncia ou solicitação foi registrada com sucesso.


## Estrutura do Projeto

* `index.html` → Estrutura das telas do sistema
* `style.css` → Estilização da interface
* `script.js` → Implementação das funcionalidades do sistema

## Objetivo

O objetivo do sistema é facilitar o descarte correto de resíduos, incentivar práticas sustentáveis e permitir que cidadãos registrem denúncias de descarte irregular em suas cidades.
